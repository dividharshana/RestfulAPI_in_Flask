# RestfulAPI_in_Flask

## Installations

* Create a virtual environment and activate
```
virtualenv -p python3 .venv
.venv/Scripts/activate
```

* Requirements.txt
```
Flask-RESTful==0.3.8
PyJWT==1.7.1
Flask-SQLAlchemy==2.4.1
```

* Install the requirements.txt
```
pip install -r requirements.txt
```

## Run the Server

```
python api.py
```
